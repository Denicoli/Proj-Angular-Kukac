# Proj-Angular-Kukac

## Passos para abrir o Back-end

- Abra o prompt na pasta `app-kukac-backend`

- Digite no prompt `npm i`

- Digite no prompt `npm run start`

## Passos para abrir o Front-end

- Abra o prompt na pasta `app-kukac`

- Digite no prompt `npm i`

- Digite no prompt `npx ng serve --aot -o`
